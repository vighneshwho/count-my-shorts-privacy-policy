# CountMyShorts Privacy Policy

Last updated: February 22, 2026

CountMyShorts is a Chrome extension that helps you track YouTube Shorts usage and show break reminders.

## Data We Handle
CountMyShorts stores the following data **locally on your device** using Chrome extension storage:
- Daily Shorts count
- Estimated watch time in milliseconds
- Session state (timestamps needed to calculate active sessions and reminders)
- User settings (reminder interval, continue target, confetti milestone)

## How Data Is Used
The data is used only to:
- Show your Shorts count and watch-time related reminders
- Preserve your extension preferences
- Reset/maintain daily stats and reminder behavior

## Data Sharing
- We do **not** sell data.
- We do **not** transfer data to our servers.
- We do **not** use third-party analytics or advertising SDKs.

## Permissions
CountMyShorts uses these Chrome permissions:
- `storage`: save stats and settings locally
- `alarms`: trigger reminder timing
- Host access to YouTube Shorts URLs: needed to detect Shorts pages and show overlays/reminders

## Data Retention and Deletion
- Data remains in local extension storage until you clear extension data or uninstall the extension.
- Uninstalling the extension removes its local data according to Chrome's extension storage behavior.

## Children's Privacy
CountMyShorts is not designed for children under 13.
